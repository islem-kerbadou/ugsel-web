Entité représentant un type de sport dans le système applicatif. 

Ce fichier définit l\textquotesingle{}entité Doctrine {\ttfamily Sport\+Type}, exposée en tant que ressource API via API Platform. Elle permet de classifier les sports selon un code unique et un libellé descriptif.

Cette entité constitue un élément structurant du modèle métier \+: elle est utilisée notamment comme entité de référence dans des relations (ex. \+: association Many\+To\+One avec l’entité {\ttfamily Sport}).

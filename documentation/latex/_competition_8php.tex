\doxysubsection{Description détaillée}
Entité représentant une compétition dans le système applicatif. 

Ce fichier définit l\textquotesingle{}entité Doctrine {\ttfamily Competition}, exposée en tant que ressource API via API Platform. Elle modélise une compétition identifiable par un identifiant unique et caractérisée par un nom.

Cette classe est persistée en base de données grâce à Doctrine ORM et automatiquement exposée sous forme d’\+API REST par API Platform.

PHP version 8.\+3

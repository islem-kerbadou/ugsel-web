\doxysubsection{Description détaillée}
Entité représentant un championnat dans le système applicatif. 

Ce fichier définit l\textquotesingle{}entité Doctrine {\ttfamily Championship}, exposée en tant que ressource API via API Platform. Elle modélise un championnat identifié de manière unique et caractérisé par un nom.

Cette classe est persistée en base de données grâce à Doctrine ORM et automatiquement exposée sous forme d’\+API REST grâce à l’attribut {\ttfamily Api\+Resource}.

PHP version 8.\+3

Description détaillée

Entité représentant un type de sport dans le système applicatif.

Ce fichier définit l'entité Doctrine SportType, exposée en tant que ressource API via API Platform. Elle permet de classifier les sports selon un code unique et un libellé descriptif.

Cette entité constitue un élément structurant du modèle métier : elle est utilisée notamment comme entité de référence dans des relations (ex. : association ManyToOne avec l’entité Sport).
